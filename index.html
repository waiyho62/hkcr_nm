Welcome to the HKCR NM Online MCQ Templates

This site provides online templates for writing MCQs for:

  1. First NM Examination - Physics Paper
  2. First NM Examination - Clinical Paper
  3. Final NM Examination

These templates are client-side programs. The actual writing of MCQs occurs in the browser of your own computer and no information will be leaked to the internet when using these templates.

Instructions:

  1. Click one of the link above to go to the online MCQ template for that paper.
  2. Fill in the MCQ template form.
  3. Author name and keyword are selectable from the drop-down lists. You can also manually type in your own if necessary.
  4. Add an image to a question if necessary. Loading a second image will overwrite the first one. If you subsequently decide not to use an image for that question, click the [Remove Image] button.
  5. "a. ", "b. ", etc are not needed for the choices. They will be added automatically by the template.
  6. Set "T/F" for choices for the First Examination or "T" for one choice for the Final Examination, by clicking the buttons [Choice A] to [Choice E].
  7. Forbidden characters will be changed automatically by the templates. These include ";" and "new paragraph" for the question, ";" for the choices, and "," and ";" for the keyword.
  8. Press the button [Save Question] to save a question. If one or more required fields have not been filled, the template will prompt you to do so.
  9. After saving a question, the question number will be incremented by one automatically. You can proceed to write another question. Please noted that the inputs to the fields will not be deleted and they can be reused if appropriate.
  10. After completion of writing all questions, press the button [Prepare Zip File] to generate a zip file containing the saved questions and the images. Then click on the link "Download Zip File" to download the zip file (ToHKCR.zip) for submission to the HKCR.
  11. After downloading the zip file, either reload the page (all fields clear) or press the button [Reset to Question 1] (all fields stay) to continue to write another set of questions.
  12. Please noted that if you leave the page before downloading the zip file, all written MCQs will be lost.
  13. The downloaded zip file is not password-protected. For security reason, you may consider to unzip it and add a password before sending through email.
