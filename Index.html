Welcome to the HKCR NM Online Question Templates

This site provides online templates for writing questions for:

  1. First NM Examination - Physics MCQ Paper
  2. First NM Examination - Clinical MCQ Paper
  3. Final NM Examination - MCQ Papers
  4. Final NM Examination - Reporting Session Paper

These templates are client-side programs. The actual writing of questions occurs in the browser of your own computer and no information will be leaked to the internet when using these templates.

Please refer to the following document for information on writing Reporting Session questions.

Format for submission of NM reporting session questions

Instructions:

  1. Click one of the link above to go to the online question template for that paper.
  2. Fill in the question template form.
  3. Author name and keyword (for MCQ only) are selectable from the drop-down lists. You can also manually type in your own if necessary.
  4. Add an image to an MCQ if necessary. For a Reporting Session question, one image is mandatory and up to four images can be added to each queston. Loading a second image will overwrite the first one. If you subsequently decide not to use an image for that question, click the [Remove Image] button.
  5. "a. ", "b. ", etc are not needed for the choices for an MCQ or short questions in a Reporting Session question . They will be added automatically by the template.
  6. Set "T/F" for choices for the First Examination or "T" for one choice for the Final Examination, by clicking the buttons [Choice A] to [Choice E].
  7. Forbidden characters will be changed automatically by the templates. These include ";" and "new paragraph" for the question in MCQ or clinical history for Reporting Session question, ";" for the choices in MCQ or short questions in Reporting Session question, and "," and ";" for the keyword.
  8. Press the button [Save Question] to save a question. If one or more required fields have not been filled, the template will prompt you to do so.
  9. After saving a question, the question number will be incremented by one automatically. You can proceed to write another question. Please noted that the inputs to the fields will not be deleted and they can be reused if appropriate.
  10. After completion of writing all questions, press the button [Prepare Zip File] to generate a zip file containing the saved questions and the images. Then click on the link "Download Zip File" to download the zip file (ToHKCR.zip) for submission to the HKCR.
  11. After downloading the zip file, either reload the page (all fields clear) or press the button [Reset to Question 1] (all fields stay - for MCQ templates only) to continue to write another set of questions.
  12. Please noted that if you leave the page before downloading the zip file, all written questions will be lost.
  13. The downloaded zip file is not password-protected. For security reason, you may consider to unzip it and add a password before sending through email.
